# @sdkwork/generations-pc-studio

Cross-modality generation studio UI primitives for PC playground surfaces:

- `SdkworkGenerationModePopupBase` — settings bar + popup shell for image/video panels
- `SdkworkStudioGenerationBottomBar` — fixed credit + generate action bar
- `formatGenerationCreditPoints` — locale-aware credit display via `@sdkwork/utils`

Import from `@sdkwork/generations-pc-studio/react` in modality packages (`image`, `video`, `music`, `audio`).
