# Examples Directory

## Purpose

Runnable examples, integration examples, sample configs, and SDK/API usage examples.

## Owner

sdkwork-generations

## Allowed Content

- Runnable example code
- Integration examples
- Sample configurations
- SDK/API usage examples
- Example documentation

- Runtime data
- Secrets or credentials
